## このサイトについて

このサイトはNext.js 15と MDXで構築したポートフォリオです。
記事の追加は `content/articles/` に `.mdx` ファイルを置くだけで反映されます。

## 記事の書き方

```
---
title: "記事のタイトル"
date: "2026-06-05"
tags: ["タグA", "タグB"]
summary: "1〜2文の要約"
cover: "/images/articles/foo/cover.jpg"  # 任意
draft: true                                # 任意：公開しない
---

本文は Markdown で記述できます。
コードブロックや見出し、リストも普通に書けます。
```

## できること

- Markdown（見出し・箇条書き・引用・コードブロック）
- 画像挿入：`public/images/articles/{slug}/` に置いて `![](/images/articles/foo/bar.jpg)`
- React コンポーネントの埋め込み（必要になったら）
